Subtitles

Yet another srt subtitle editor.

The main goal of this editor was to to have a broad constraint system for subtitles that would allow an easy and fast workflow. A great set of shortcuts also helps you get done earlier. Since this project mainly focuses on these constraints it is otherwise very minimalistic.

You can download the App on www.eniestudio.de
